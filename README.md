# Dialogue Tree Visualizer

A tool for visualizing and editing dialogue trees used in video games.  
Built with Python - supports loading and saving dialogue structures in JSON format.

## Motivation

This project was born from a deep interest in game development and narrative systems.  
Dialogue trees are at the heart of story-driven games, and understanding how they work  
from the inside is essential for anyone who wants to contribute to that craft.  
Inspired by the narrative depth of projects like The Witcher series by CD Projekt Red.

## Tech Stack

- **Language:** Python
- **UI Library:** Tkinter / PyQt (to be decided)
- **Data Format:** JSON
